### API ENDPOINTS

- GET `todoapi/todos` get all todos
- GET `todoapi/todos/:id` get 'Todo' by id
- POST `todoapi/todos` add new 'Todo'
- PUT `todoapi/todos/:id` update 'Todo' by id
- DELETE `todoapi/todos/:id` remove 'Todo' by id
